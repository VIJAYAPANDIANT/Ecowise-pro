## 🎯 Mission Statement

**EcoWise Pro++** is a next-generation decision support system designed to bridge the gap between human consumption and environmental accountability. By leveraging rule-based AI logic, it provides real-time, explainable guidance on waste disposal, reducing carbon footprints one item at a time.

---

## 🧠 Core Intelligence Logic

Unlike opaque "black box" models, EcoWise Pro++ utilizes a **Transparent Reasoning Engine**.

1.  **Input Parsing**: Natural language keywords are extracted and normalized.
2.  **Taxonomy Matching**: Inputs are scored against a verified environmental database.
3.  **Risk Calibration**: Disposal advice is weighted by toxicity (e.g., _Biomedical > High Priority_).
4.  **Impact Quantification**: Estimated CO2 savings are calculated per interaction.

## 🚀 Installation & Usage

1.  Clone the repository.
2.  Open `index.html` in any modern web browser.
3.  Type a waste item (e.g., _"old laptop battery"_) into the console.
4.  Receive immediate, safety-critical disposal instructions.
